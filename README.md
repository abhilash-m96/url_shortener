# URL shortener

## Setup:
```bash=
pip3 install -r requirements.txt
```

**Edit the .env configuration according to your setup**
```
DEBUG=true
PORT=5000
MONGO_HOST=localhost
USER_NAME=""
PASSWORD=""
DATABASE="url_shortener"
MONGO_PORT=27017
```

**Running the service:**
```bash=
cd src/
python3 app.py
```
